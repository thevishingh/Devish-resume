# 📄 Vishal Singh Digital Resume

A modern digital resume built with Next.js, TypeScript, and Tailwind CSS.

This project showcases my professional experience, technical skills, career highlights, and contact information in a clean, responsive format. It is designed to be simple, professional, and easy to navigate across devices.

## ✨ Features

- Responsive resume layout
- Dark and light mode
- Downloadable PDF resume
- Skills and experience highlights
- Clean, modern UI

## 🛠️ Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/thevishingh/Devish-resume.git
   ```

2. Navigate to the project folder:
   ```bash
   cd Devish-resume
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open `http://localhost:3000` in your browser.
